대스타 1차 멘토링 - 박재범(LG CNS)

1. 학습 대상 : 종자 이미지 --> 데이터 현황 파악!
   1.1 클래스의 개수 약 300개 --> 데이터 증강 필수 (ex. 반전, 회전, 뒤집기 등)
   1.2 종자별 이미지 개수가 다름 --> 클래스 당 데이터 비율 균일하게 하기
	(ex. 이미지 분할 --> 여러개를 한꺼번에 찍은 사진 분할)
   1.3 회사 내 종자 데이터나 학습 모델이 있으면 활용가능
   1.4 추가 데이터 촬영 가능
   
2. 사전 학습 모델 찾아보기(ResNet 등)

3. 베이스 모델 만들고 클래스당 편차 확인(어느 종자를 더 잘 맞추고 못맞추는지)

4. 평가
   4.1 1차평가 : 10월 쯤 --> 분류 정확도 위주
   4.2 2차평가 : 어떤 형식의 제출인지 알아보기(ex. 앱 프로토타입, 분류정확도 등)
	실제로 수목원에서는 앱개발을 염두해두고 있음

위의 평가를 정확하게 알아보고 일정 계획 세우기!
